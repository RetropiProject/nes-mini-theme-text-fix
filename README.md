# Retropie es-theme-nes-mini fix (text decay)
This small script fixes a text alignment issue on certain screens in the Retropie theme es-theme-nes-mini. It adjusts the display so the text is properly aligned in the game selection menu without manual theme edits. This is a simple solution for users experiencing text offset problems on some setups

# Installation
Type the following (case-sensitive) line of code into the Raspberry Pi terminal :

wget -O - "https://raw.githubusercontent.com/RetropiProject/nes-mini-theme-text-fix/main/fix_nes_mini_theme.sh" | sudo bash
